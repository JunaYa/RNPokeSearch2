# RNPokeSearch

A simple Pokemon search app demo for getting started with React Native.

## How to run

1.  Clone the repo:

```
git clone https://github.com/anchetaWern/RNPokeSearch2.git RNPokeSearch
cd RNPokeSearch
```

2.  Install the dependencies:

```
yarn
```

3.  Run the app:

```
yarn start
```
